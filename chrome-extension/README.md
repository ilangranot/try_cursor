## Installation

### Method 1: Install from Website
Visit https://ilangranot.github.io/try_cursor and click the "Download Extension" button.

### Method 2: Manual Installation

1. Download the extension files
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable "Developer mode" (toggle in top right)
4. Click "Load unpacked"
5. Select the `chrome-extension` folder
6. The extension is now installed!

## Setup

1. Click the extension icon in your Chrome toolbar
2. Enter your OpenAI API key (get one at https://platform.openai.com/api-keys)
3. Click "Save API Key"
4. Start chatting with ChatGPT!

## Usage

- Click the extension icon to open the popup
- Type your question or use quick actions
- Get instant AI responses
- Quick actions include:
  - Summarize current page
  - Explain page in simple terms
  - Get key points from page

## Privacy

- Your API key is stored locally in Chrome's sync storage
- No data is sent to any server except OpenAI's API
- Page content is only accessed when you use quick actions

## Requirements

- Chrome browser (version 88 or later)
- OpenAI API key (get one at https://platform.openai.com/api-keys)
