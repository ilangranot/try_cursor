# GitHub Pages Setup for try_cursor

## Step 1: Create Repository on GitHub

1. Visit: https://github.com/new
2. Repository name: **try_cursor**
3. Description: (optional) Modern responsive website
4. Make it **PUBLIC** (required for free GitHub Pages)
5. **DO NOT** check "Add a README file"
6. **DO NOT** check "Add .gitignore"
7. **DO NOT** check "Choose a license"
8. Click **"Create repository"**

## Step 2: Connect and Push (Run these commands)

After creating the repository, run these commands in your terminal:

```bash
git remote add origin https://github.com/ilangranot/try_cursor.git
git push -u origin main
```

## Step 3: Enable GitHub Pages

1. Go to: https://github.com/ilangranot/try_cursor/settings/pages
2. Under "Source", select **"Deploy from a branch"**
3. Branch: **main**
4. Folder: **/** (root)
5. Click **"Save"**

## Step 4: Your Website URL

Your website will be live at:
**https://ilangranot.github.io/try_cursor**

(It may take 1-2 minutes for the site to go live after enabling Pages)
